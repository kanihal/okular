Carlito is metric-compatible with Microsoft Calibri. It is bundled so that
PDFs using non-embedded Calibri render with correct glyph spacing on systems
where neither Calibri nor Carlito is installed (fontconfig's
30-metric-aliases.conf maps Calibri to Carlito).
